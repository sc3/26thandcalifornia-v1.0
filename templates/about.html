Disclaimer

The collected inmate data is known to be inaccurate, both at the source, the Cook County Jail, and the way the data is captured is not guaranteed to be fault free. Therefore do not take the data or statistics shown as definitive, rather, viewed them as suggestive.

A Note on Execution Performance

To generate the Stats views, the entire data set is loaded the first time you request one of these views. The loading of this data takes some time, upwards of a minute. Subsequent views should render much quicker.

Rendering some views requires a lot of calculations and the plotting of a lot of data points or elements, so some of them take a bit of time to be displayed. We intend to reduce these delays.
